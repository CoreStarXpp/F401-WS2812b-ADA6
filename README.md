#### 项目简介
以最小的资源消耗和最高的代码密度实现远程蓝牙控制灯光效果

#### 软件架构
采用纯汇编完成，附带关键函数动态链接和配套源码（可用高级语言如c/c++调用）

#### 安装教程

1.  拉取源码
2.  安装arm-none-eabi
3.  编译
4.  组装配件
5.  烧录与测试

#### 使用说明

1.  纯汇编源码，带精细注释，配套B站讲解视频
2.  测试平台STM32F401RCT6开发板
3.  符合arm-none-eabi要求
4.  高级语言调用请参阅改版源码
